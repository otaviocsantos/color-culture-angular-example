How Color Culture works

A culture is like a color palette, it holds colors together:

The main difference is that in a culture you may have dynamic collors.

This is a culture:

+

It has a color (yellow) and a modifier: {{cmykPicker.color.toString()}}, Color Culture then takes the first color, adds the modifier to it and generates the second color.

That means that if you alter the first color or the modfier, the second color will be automatically updated.

You are not limited to CMYK, you may use RGB, HSL, LAB, XYZ or create your own color model.
